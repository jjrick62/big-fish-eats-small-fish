# 大鱼吃小鱼

> 经典休闲游戏 — Python + Pygame 实现

## 玩法

鼠标控制鱼在海中游动：
- **吃小鱼** — 碰到比自己小的鱼，吃掉后体型增长
- **躲大鱼** — 碰到比自己大的鱼，游戏结束
- **成长系统** — 分数随吞食积累，体积逐步变大

## 技术栈

| 层 | 技术 |
|---|---|
| 语言 | Python 3 |
| 游戏引擎 | Pygame |
| 图形 | 原生绘制（圆+多边形鱼） |
| 字体 | 系统黑体（SimHei） |

## 快速开始

```bash
pip install pygame
python fish_game.py
```

或双击 `启动游戏.bat`

## 游戏机制

- 鱼群随机生成，含大小不同的 AI 鱼
- AI 鱼有自主游动轨迹，大追小逃
- 玩家鱼尺寸随分数动态增长
- 屏幕边缘环绕（超出自动从另一侧出现）

## 操作

- **鼠标移动** — 控制鱼游动方向
- **鱼自动跟随鼠标位置**
